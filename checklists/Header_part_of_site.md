### Навигационная панель
- [x] Отображаются все пункты меню (Акции, Контакты, Доставка и оплата, Статьи, Отзывы)
- [x] Все пункты кликабельны и ведут на соответствующие страницы
- [x] Логотип ведёт на главную страницу

### Выбор города
- [x] Отображается текущий город (Казань)
- [x] Выпадающий список раскрывается при клике
- [x] Выбранный город сохраняется и отображается

### Профиль пользователя
- [x] Иконка профиля отображается (авторизированный пользователь)
- [x] Переход на страницу профиля при нажатии на иконку (/profile) (авторизированный пользователь)
- [x] Отображается кнопка "Войти" (неавторизированный пользователь)
- [x] Переход на страницу авторизации при нажатии на кнопку "Войти" (неавторизированный пользователь)
 
### Поиск
- [x] Поиск по существующему товару → результаты
- [x] Поиск по несуществующему → «Ничего не найдено»

- [x] Поиск по цене реализован

### Категории
- [x] Все 6 категорий отображаются
- [x] Каждая категория ведёт на свою страницу
- [x] Активная категория визуально выделена

### Сообщение о предзаказе
- [x] Отображается с 19:31 до 08:59 (МСК)
