## Навигационная панель
- [x] Отображаются все пункты меню: «Акции», «Контакты», «Доставка и оплата», «Статьи», «Отзывы»
- [x] Все пункты кликабельны и ведут на соответствующие страницы:<br>«Акции» `/actions`, «Контакты» `/info/12889`, «Доставка и оплата» `/info/12890`, «Статьи» `/articles`, «Отзывы» `/reviews`
- [x] Логотип ведёт на главную страницу (`/`)

---

## Выбор города
- [x] Отображается текущий город («Казань» по умолчанию)
- [x] Выпадающий список раскрывается при клике
- [x] При выборе другого города он сохраняется и отображается как текущий
- [x] При смене города осуществляется редирект на страницу, соответствующую данному городу <br>(пример: город Самара - `https://vremya-pirogov-samara.ru`)

---

## Профиль пользователя

### Авторизованный пользователь
- [x] Иконка профиля отображается корректно (см. `/screenshots/unauthorized_profile.png`)
- [x] При клике на иконку выполняется переход на страницу `/profile`
- [x] В профиле отображаются корректные данные: телефон, имя, email, дата рождения, баллы, количество заказов
- [x] В header части сайта, справа от иконки профиля отображается кол-во бонусных баллов пользователя ( пример: `/screenshots/header_bonus_points.png`)

### Неавторизованный пользователь
- [x] Отображается кнопка «Войти»
- [x] При клике на «Войти» появляется модальное окно (без перезагрузки)
- [x] После успешного входа иконка профиля заменяет кнопку «Войти»
- [x] Если неавторизованный пользователь пытается открыть страницу профиля `/profile`, то у него открывается главная страница сайта

## Поиск
- [x] Поле поиска с плейсхолдером «Поиск по меню» отображается
- [x] Поиск по существующему товару → отображаются релевантные результаты
- [x] Поиск по несуществующему товару → отображается «К сожалению, ничего не найдено»
- [x] Поиск с пустой строкой → показываются все товары
- [x] Поиск игнорирует лишние пробелы в начале/конце строки
- [x] Поиск регистронезависимый («Пирог» и «пирог» дают одинаковые результаты)

---

## Категории товаров
- [x] Отображаются все 6 категорий: «Сытные пироги», «Осетинские пироги», «Сладкие пироги», «Десерты», «Напитки», «Сеты»
- [x] Каждая категория кликабельна и ведёт на соответствующую страницу каталога:<br> «Сытные пироги» `/catalog/sitnie-pirogi`, «Осетинские пироги» `/catalog/osetinskie-pirogi`<br>«Сладкие пироги» `/catalog/sladkie-pirogi`, «Десерты» `/catalog/deserti`, «Напитки» `/catalog/napitki`, «Сеты» `/catalog/seti`
- [x] Активная категория визуально выделена (background-color: `#f89b34`; color: `#fff`;)
- [x] При переходе на страницу категории, соответствующая категория подсвечивается

## Сообщение о предзаказе
- [x] Отображается в нерабочее время: с 19:31 до 08:59 (МСК)
- [x] Не отображается в рабочее время: с 09:00 до 19:30 (МСК)
- [x] Текст сообщения: *«Мы начнем работать с 09:00, но вы можете оставить предзаказ!»* — соответствует макету
- [x] Сообщение не содержит орфографических и грамматических ошибок
