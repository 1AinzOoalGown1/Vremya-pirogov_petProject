- `/screenshots/header_testit_checklist.png` — Чек-лист в системе TestIT

---

## Навигационная панель
- [x] Отображаются все пункты меню: «Акции», «Контакты», «Доставка и оплата», «Статьи», «Отзывы»
- [x] Все пункты кликабельны и ведут на соответствующие страницы
- [x] Логотип ведёт на главную страницу (`/`)

---

## Выбор города
- [x] Отображается текущий город («Казань» по умолчанию)
- [x] Выпадающий список раскрывается при клике
- [x] При выборе другого города он сохраняется и отображается как текущий
- [x] При смене города обновляется информация о доставке (цены/условия)

---

## Профиль пользователя

### Авторизованный пользователь
- [x] Иконка профиля отображается в хедере
- [x] При клике на иконку выполняется переход на страницу `/profile`
- [x] В профиле отображаются корректные данные: имя, телефон, email, баллы, количество заказов

### Неавторизованный пользователь
- [x] Отображается кнопка «Войти»
- [x] При клике на «Войти» выполняется переход на страницу авторизации
- [x] После успешного входа иконка профиля заменяет кнопку «Войти»

---

## Поиск
- [x] Поле поиска с плейсхолдером «Поиск по меню» отображается
- [x] Поиск по существующему товару → отображаются релевантные результаты
- [x] Поиск по несуществующему товару → отображается «Ничего не найдено»
- [x] Поиск с пустой строкой → показываются все товары
- [x] Поиск игнорирует лишние пробелы в начале/конце строки
- [x] Поиск регистронезависимый («Пирог» и «пирог» дают одинаковые результаты)

---

## Категории товаров
- [x] Отображаются все 6 категорий: «Сытные пироги», «Осетинские пироги», «Сладкие пироги», «Десерты», «Напитки», «Сеты»
- [x] Каждая категория кликабельна и ведёт на соответствующую страницу каталога
- [x] Активная категория визуально выделена (цветом, подчёркиванием, жирным шрифтом)
- [x] При переходе на страницу категории, соответствующая категория подсвечивается
---

## Сообщение о предзаказе
- [x] Отображается в нерабочее время: с 19:31 до 08:59 (МСК)
- [x] Не отображается в рабочее время: с 09:00 до 19:30 (МСК)
- [x] Текст сообщения: *«Мы начнем работать с 09:00, но вы можете оставить предзаказ!»* — соответствует макету
- [x] Сообщение не содержит орфографических и грамматических ошибок
