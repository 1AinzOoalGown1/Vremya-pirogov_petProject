## Открытие и закрытие модального окна

- [x] При клике на кнопку «Войти» (неавторизованный пользователь) открывается модальное окно авторизации
- [x] Модальное окно содержит заголовок «Вход»
- [x] Модальное окно закрывается при клике на крестик (×) в правом верхнем углу
- [x] Модальное окно закрывается при клике на любое место вне модального окна (на оверлей)
- [x] После закрытия окна состояние страницы не меняется (пользователь остаётся неавторизованным)

## Согласие на обработку персональных данных

- [x] Отображается чекбокс с текстом: *«Я даю согласие на обработку моих персональных данных, в соответствии с Федеральным законом от 27.07.2006 г. №152-ФЗ "О персональных данных", на условиях, определенных политикой в области обработки и обеспечения безопасности персональных данных»*
- [x] Текст «политикой» | «согласие» является кликабельной ссылкой на страницу `/privacy` | `/agreement`
- [x] Чекбокс неактивен (не отмечен) по умолчанию
- [x] Чекбокс можно отметить/снять кликом

### Валидация согласия
- [x] При клике на любую кнопку авторизации («Войти через Telegram», «Войти через Max», «Войти через VK», «Войти через Яндекс», «Войти по номеру телефона») с **неотмеченным** чекбоксом под чекбоксом появляется сообщение: *«Чтобы продолжить, установите этот флажок.»*, редирект на страницу авторизации не осуществляется
- [x] При клике на любую кнопку авторизации с **отмеченным** чекбоксом сообщение об ошибке **не** отображается
- [x] При повторном клике на кнопку авторизации после установки флажка сообщение об ошибке исчезает

## Основное окно авторизации (способы входа)

- [x] Отображаются кнопки: «Войти через Telegram», «Войти через Max», «Войти через VK», «Войти через Яндекс»
- [x] У каждой кнопки отображается соответствующий логотип/иконка социальной сети/мессенджера
- [x] Отображается текст «Другие способы авторизации» (кликабельный)
- [x] При нажатии на «Другие способы авторизации» появляется надпись (постоянная) «ИЛИ» и кнопка «Войти по номеру телефона»

---

## Вход через Telegram

- [x] При клике на «Войти через Telegram» с отмеченным чекбоксом открывается новое окно/вкладка с авторизацией через Telegram (бота)
- [x] На странице сайта вместо модального окна с вариантами авторизации появляется окно ожидания
- [x] В окне ожидания отображается крутящийся спиннер (кружок загрузки)
- [x] В окне ожидания отображается текст: *«Пытаемся авторизовать Вас... Откройте мессенджер Telegram. В чате с ботом авторизации нажмите кнопку "Поделиться контактом"»*
- [x] Кнопка «Поделиться контактом» открывает новое окно/вкладку с страницей авторизации через Telegram
- [x] В окне ожидания отображается кнопка «Вернуться назад», при клике на которую возвращается модальное окно авторизации
- [x] В окне ожидания отображается крестик (×), при клике на который окно ожидания закрывается (модальное окно авторизации не появляется)
- [x] При успешной авторизации через Telegram модальное окно закрывается, кнопка «Войти» заменяется на иконку профиля
- [x] При клике на иконку профиля выполняется переход на страницу `/profile`

---

## Вход по номеру телефона

### Открытие окна ввода номера
- [x] При клике на «Войти по номеру телефона» (с отмеченным чекбоксом) отображается новое состояние модального окна
- [x] В окне отображаются кнопки: «Войти через Telegram», «Войти через Max», «Войти через Яндекс» (VK отсутствует)
- [x] Отображается текст: *«Подтвердите номер телефона. Чтобы войти или зарегистрироваться»*
- [x] Отображается поле для ввода номера телефона

### Поле ввода номера телефона
- [x] В поле можно вводить **только цифры** (буквы и спецсимволы не вводятся)
- [x] Ограничение на ввод — **10 цифр** (не более)
- [x] Маска ввода: `+7 (xxx) xxx-xx-xx` или аналогичный формат

### Кнопка «Получить код для входа»
- [x] Отображается кнопка «Получить код для входа»
- [x] При клике на кнопку с **пустым** полем номера появляется сообщение: *«Заполните это поле»*
- [x] При клике на кнопку с **неполным** номером (менее 10 цифр) появляется сообщение: *«Некорректный номер телефона»*
- [x] При клике на кнопку с **корректным** номером (10 цифр) отображается новый экран с полем для ввода кода
- [x] В новом экране отображается текст: *«На номер +7 (xxx) xxx-xx-xx отправлен код подтверждения. Введите его в поле ниже»*
- [x] Номер телефона в тексте соответствует введённому ранее

### Поле ввода кода подтверждения
- [x] Отображается поле из 4 квадратиков для ввода кода
- [x] В поле можно вводить **только цифры** (буквы и спецсимволы не вводятся)
- [x] Автоматический переход к следующему квадратику после ввода цифры
- [x] Отображается кнопка «Вернуться назад»
- [x] При клике на «Вернуться назад» возвращается предыдущее состояние окна (ввод номера телефона)
- [x] Введённый номер телефона сохраняется при возврате

### Кнопка «Войти» (подтверждение кода)
- [x] Отображается кнопка «Войти»
- [x] При вводе **корректного** кода выполняется успешная авторизация
- [x] При успешной авторизации модальное окно закрывается, кнопка «Войти» заменяется на иконку профиля
- [x] При клике на иконку профиля выполняется переход на страницу `/profile`
- [x] При вводе **некорректного** кода появляется сообщение: *«Не удалось авторизоваться. Код неверный»*
- [x] Сообщение об ошибке отображается под полем ввода кода
